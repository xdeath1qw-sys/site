# 🎯 Автоматическое перенаправление после Veto

## Что было сделано

После завершения процесса выбора карт (veto), пользователи автоматически перенаправляются на специальную страницу с сообщением о завершении.

## ✨ Основные изменения

### 1. **Автоматический редирект**
В `js/veto.js` добавлен автоматический редирект после завершения вето:
```javascript
if (veto.status === 'done') {
  // Через 1.5 секунды перенаправляем на страницу завершения
  setTimeout(() => {
    window.location.href = `veto-done.html?map=${mapName}&vetoId=${vetoId}`;
  }, 1500);
}
```

### 2. **Улучшенная страница veto-done.html**

#### Визуальные улучшения:
- ✅ **Анимированная иконка** с эффектом float
- 🎨 **Градиентный заголовок** с анимацией появления
- 💫 **Пульсирующая карточка** с эффектом pulse
- 🎯 **Анимированная кнопка** xplay с постоянной пульсацией

#### Функциональность:
- 📋 **Показ всех выбранных карт** (не только финальной)
- 🔊 **Звуковое уведомление** при открытии страницы
- ⏱️ **Таймер 20 секунд** до автоматического редиректа
- 🛑 **Остановка таймера** при клике на любую ссылку
- 🎮 **Большая кнопка xplay.gg** для перехода в игру

### 3. **Отображение информации**

Страница показывает:
```
✅ Выбор карт окончен!

Все этапы вето завершены. Карты выбраны.
Заходите в игру через xplay и начинайте матч!

📋 Выбранные карты:
1. Mirage (Team A)
2. Inferno (Team B)
3. Dust2 (Decider)

🎮 [Открыть xplay.gg]
🏆 [К турнирам]
🏠 [На главную]

⏱️ Автоматический переход на главную через 20 сек
```

## 🎨 Анимации

### Иконка
- Появление с эффектом bounce
- Постоянное плавание вверх-вниз (float)

### Заголовок и текст
- Появление снизу вверх с задержкой (staggered animation)

### Карточка с картами
- Пульсация с эффектом свечения границ

### Кнопка xplay
- Постоянная лёгкая пульсация (привлекает внимание)
- Увеличение при наведении

## 🔊 Звуковое уведомление

При открытии страницы воспроизводится короткий звуковой сигнал:
- Частота: 800 Hz
- Длительность: 0.5 секунды
- Тип: синусоида
- Громкость: 30%

## ⚙️ Параметры URL

Страница принимает параметры:
- `map` - название финальной карты (для совместимости)
- `vetoId` - ID вето (для загрузки полной информации)

Пример:
```
veto-done.html?map=Dust2&vetoId=1234567890
```

## 🎯 Как это работает

1. **Завершение вето** в veto.js
2. **Задержка 1.5 сек** (для восприятия последнего действия)
3. **Редирект на veto-done.html** с параметрами
4. **Загрузка данных вето** из localStorage
5. **Отображение всех карт** с командами/decider
6. **Воспроизведение звука** (если доступно)
7. **Запуск таймера** обратного отсчёта (20 сек)
8. **Автоматический возврат** на главную (если не перешли по ссылкам)

## 📱 Адаптивность

Страница адаптивна для всех устройств:
- Десктоп: полноразмерная карточка
- Планшет: уменьшенные отступы
- Мобильный: компактный вид с сохранением всех функций

## 🔗 Интеграция с xplay

Большая кнопка "Открыть xplay.gg" ведёт на:
```
https://xplay.gg
```

Открывается в новой вкладке с атрибутами безопасности:
- `target="_blank"`
- `rel="noopener"`

## 🎨 Кастомизация

Все цвета и анимации используют CSS переменные из основного стиля:
- `--primary` - основной цвет
- `--accent` - акцентный цвет
- `--bg-card` - фон карточки
- `--border-active` - активная граница

## ✅ Результат

Теперь после завершения выбора карт:
1. ✅ Пользователи видят красивую страницу с результатами
2. 🎮 Есть яркая кнопка для перехода в xplay
3. 📋 Отображаются все выбранные карты
4. 🔊 Звуковое уведомление привлекает внимание
5. ⏱️ Автоматический редирект если пользователь не действует
